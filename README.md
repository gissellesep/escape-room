# Requisitos

- XAMPP
- PHP 8+
- MySQL

# Instalación

1. Descargar zip del repositorio
2. Copiar carpeta descomprimida en htdocs
3. Importar database.sql en phpMyAdmin http://localhost/phpmyadmin/ 
4. Abrir http://localhost/escape-room/
